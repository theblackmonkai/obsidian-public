# Platform Formatting Guide

How to format messages for different platforms.

---

## Telegram

- Tables: every line under 25 chars
- No markdown tables - use plain text
- Headers: **bold** or CAPS
- Links: auto-detected

---

## Discord

- No markdown tables - use bullet lists
- Links: wrap in <> to suppress embeds: <https://example.com>
- Code: single backticks for inline, triple for blocks

---

## WhatsApp

- No headers - use **bold** for emphasis
- No markdown at all really
- Plain text works best
