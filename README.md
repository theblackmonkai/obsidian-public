# Public Vault

Public notes anyone can view.

## Setup
1. Clone: git clone https://github.com/theblackmonkai/obsidian-public.git
2. Open as vault in Obsidian
3. **When asked about .obsidian: Say YES** - keeps settings synced across devices
4. Run: git config core.hooksPath .githooks

## Security
Pre-push hook scans for personal info before pushing.
